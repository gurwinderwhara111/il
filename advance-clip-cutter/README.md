# Advance Clip Cutter

Advance Clip Cutter is a Flask web app for uploading videos, generating numbered vertical clips, and burning titles into each clip.

Start here:

- Full guide: `docs/user-guide/README.md`
- Setup: `docs/user-guide/SETUP.md`
- Product requirements: `docs/user-guide/PRD.md`
- Project structure: `docs/user-guide/PROJECT_STRUCTURE.md`

## Quick Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
http://127.0.0.1:3000
```

The active app is `app.py` plus files in `public/`.

From the repository root, `python app.py` still forwards here through a compatibility launcher.
